# PL0-interpreter
simple pl0 interpreter with flex, bison, gcc

## Build and run
```bash
### build
$ flex pl0.l
$ bison pl0.y
$ gcc -o pl0 pl0.tab.c -ly -ll
### run
$ pl0 <sample.pl0
```

### sample.pl0
```
const m=20, n=7;
var x,y,z;
procedure multiply;
	var a,b;
begin a:=x; b:=y; z:=0;
	while b>0 do
	begin
		if odd b then z:=z+a;
		a:=2*a; b:=b/2;
	end
end;
begin
	x:=m; y:=n; call multiply;
end.
```
